- ✅ 图片加载显示提示"点击图片可放大查看"
- ✅ 点击图片有透明度反馈
- ✅ 页面切换时自动重置为显示正面

## 🔧 技术实现

### 数据结构
每条签文包含图片路径：
```javascript
{
  "id": 1,
  "imageFront": "senso-ji-omikuji-main/1_0.jpg",
  "imageBack": "senso-ji-omikuji-main/1_1.jpg",
  ...
}
```

### 核心功能代码

#### 1. 正反面切换
```javascript
data() {
  return {
    showImageSide: 'front'  // 'front' 或 'back'
  }
}
```

#### 2. 全屏查看
```javascript
showImageFullscreen(imagePath) {
    this.fullscreenImage = 'data/' + imagePath;
}

closeFullscreen() {
    this.fullscreenImage = null;
}
```

#### 3. 图片路径
```html
<img :src="'data/' + currentFortune.imageFront" />
```

## 📱 使用说明

### 查看签文图片
1. 抽签后进入签文详解页
2. 在签号和签级下方会看到**【浅草寺原版签文】**区域
3. 默认显示**正面**

### 切换正反面
1. 点击"正面"或"背面"标签按钮
2. 图片会立即切换
3. 当前查看的一面会高亮显示

### 放大查看
1. 点击任意一张图片
2. 图片会全屏显示
3. 点击图片外区域或右上角的"×"关闭

## 📊 图片资源

### 图片来源
- 所有100条签的图片来自浅草寺实拍
- 位置：`data/senso-ji-omikuji-main/`
- 命名规则：`{签号}_0.jpg`（正面）、`{签号}_1.jpg`（背面）

### 图片内容

- 显示签号（如"第一 大吉"）
- 显示签诗（日文汉字）
- 显示各项运势（如"愿望、疾病、遗失物"等）

#### 背面（_1.jpg）
- 显示详细解说（日文）
- 显示英文翻译
- 显示典故说明
- 来自浅草寺官方
