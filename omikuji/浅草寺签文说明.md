# 浅草寺御神签（Senso-ji Omikuji）

## 📖 关于浅草寺签文

浅草寺（Senso-ji）位于日本东京，是东京最古老、最著名的寺庙之一。浅草寺的御神签（おみくじ）系统是日本传统占卜文化的代表。

## 🎲 签级体系

浅草寺使用7级签级体系，共100签：

| 签级 | 日文 | 数量 | 占比 | 说明 |
|------|------|------|------|------|
| 大吉 | 大吉 | 17条 | 17% | 最吉利的签，运势极佳 |
| 吉 | 吉 | 35条 | 35% | 吉利的签，运势良好 |
| 半吉 | 半吉 | 5条 | 5% | 半吉半凶，需谨慎 |
| 小吉 | 小吉 | 4条 | 4% | 小有吉利，稳步前进 |
| 末吉 | 末吉 | 3条 | 3% | 平稳为主，顺其自然 |
| 凶 | 凶 | 30条 | 30% | 运势不佳，谨慎行事 |
| 大凶 | 大凶 | 6条 | 6% | 运势极差，静待转机 |

## 🔧 自定义配置

### 修改签级过滤

如果您想调整显示的签级，编辑 `js/app.js`：

```javascript
// 当前配置：只显示吉祥签
const luckyFortunes = this.allFortunes.filter(f =>
    f.level === '大吉' ||
    f.level === '吉' ||
    f.level === '半吉' ||
    f.level === '小吉' ||
    f.level === '末吉'
);
```

**其他配置方案**：

1. **超级吉祥模式**（只要大吉和吉）：
```javascript
const luckyFortunes = this.allFortunes.filter(f =>
    f.level === '大吉' || f.level === '吉'
);
```

2. **完整模式**（显示所有签，包括凶签）：
```javascript
const luckyFortunes = this.allFortunes; // 不过滤
```

3. **平衡模式**（吉祥签 + 半吉）：
```javascript
const luckyFortunes = this.allFortunes.filter(f =>
    f.level === '大吉' ||
    f.level === '吉' ||
    f.level === '半吉'
);
```

## 🙏 日本抽签文化

### 传统流程
1. 投入香钱（通常100日元）
2. 摇动签筒，抽出签棒
3. 根据签号取对应的签文纸
4. 阅读签文和解释

### 抽到凶签怎么办？
在日本传统中，抽到凶签有特殊处理方式：
- **系在寺庙指定的地方**：将凶签系在寺庙的绳索或树枝上，寓意"留下厄运"
- **不要带回家**：避免把坏运气带回家
- **重新调整心态**：把凶签当作警示，提醒自己谨慎行事

### 吉签的处理
- 可以带回家保存
- 放在钱包、包包等随身物品中
- 一年后可以送回寺庙焚化
